# TH React CRUD

Dự án React CRUD đơn giản để quản lý danh sách dữ liệu mẫu.

## Bắt đầu nhanh

```bash
git clone https://github.com/<your-org-or-user>/TH-React-CRUD.git
cd TH-React-CRUD/react-crud-app
npm install
npm run dev
```

Sau khi chạy lệnh `npm run dev`, mở trình duyệt tại địa chỉ mà Vite hiển thị (thường là http://localhost:5173).

## Thư mục chính

- `react-crud-app/src` – mã nguồn React chính.
- `react-crud-app/src/components` – các thành phần giao diện.
- `react-crud-app/src/services` – hàm gọi API hoặc xử lý dữ liệu.

## Yêu cầu

- Node.js 18+ khuyến nghị.
- npm 9+ (cài kèm với Node.js).
